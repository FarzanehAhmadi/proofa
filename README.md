# Proofa

A clean and professional website for a privacy-focused cryptography company specializing in zero-knowledge proofs, smart contract development, and Web3 privacy infrastructure.

## About

Proofa represents cryptographic proof engineering and privacy-preserving technology. The website uses a minimal dark design with subtle blue accents to reflect technical sophistication and trust.

## Features

- Modern minimal design with dark theme
- Hero section with strong technical branding
- About and philosophy section
- Services showcasing zero-knowledge proofs, smart contracts, and Web3 privacy
- Projects section with dynamic routing
- Technical blog
- DApp interaction page with wallet connection
- Contact section

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

```
